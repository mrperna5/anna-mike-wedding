# Photography

Drop real images here when you have them (engagement, the villa, Pienza, Val d'Orcia).

Sections currently use flat-tone placeholders with the `.photo` class. To swap in a real
photo, either set it as the `background` of the `.photo` block in that component, or replace
the `<figure class="photo">` with Astro's `<Image />` (`astro:assets`) for responsive,
optimised output. Keep the `<figcaption class="photo__cap">` for the italic caption.

Suggested first images: a wide Val d'Orcia / cypress shot for the hero, a candid of the two
of you for "Our story", and the villa for "Venue".
